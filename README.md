# 期货研究工具

一个可本地启动、可通过浏览器访问的中文期货研究门户原型。

## 启动方式

```bash
python3 app.py
```

启动后访问 `http://127.0.0.1:8000`。

如需安装项目依赖，直接执行：

```bash
pip install -r requirements.txt
```

当前环境里已检测到的版本只有 `pandas==2.3.3`。
`akshare`、`arch`、`statsmodels`、`yfinance` 在这台机器上还未安装，所以 `requirements.txt` 中对它们使用的是兼容下限；等你实际安装并验证后，如果你想，我可以再帮你把它们改成严格锁定版本。

## 当前功能

- 单页布局：上方建模区、下方研究区
- 建模区的`期货产品`会同时控制整页的研究对象
- 建模区可选择模型并发起本地预测任务
- 预测任务状态会显示“模型运算中”与运行计时
- 任务完成后状态会提示前往研究区检查结果
- 预测结果会写入本地 SQLite：`data/predictions.db`
- 日线行情会写入同一个 SQLite 中的 `market_data` 表
- 研究区提供日线 K 线占位图，可叠加多个模型的最近一次预测结果
- 右侧参数区是可滚动多选列表，支持 MA、布林带和模型结果勾选
- 每个模型在参数区会显示该模型最近一次更新时间

## 说明

本轮已经补上独立的预测运行器、`models/` 目录中的 ARIMA/GARCH 模型，以及 `data_extraction.py` 数据提取模块。
数据提取优先读取本地 SQLite 的日线数据，并在需要时尝试通过 AkShare 与 Yahoo Finance 刷新。

## 新增预测方法

- `多模型综合系统`
  - 参考外部多模型铜价预测仓库的算法框架，抽象成了适用于本项目所有期货品种的通用版本
  - 核心结构为：短期技术路径 + 中期宏观代理路径 + 长期基本面代理路径 + 市场状态/风险调整 + 动态加权融合
  - 不再依赖铜专用数据源或硬编码变量，而是基于本地已有的通用日线数据（价格、成交量、波动率、动量、回撤等）构建多模型综合预测
